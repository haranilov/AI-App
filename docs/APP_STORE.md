# App Store Submission Guide

Step-by-step instructions to build HookAI and submit it to the Apple App Store.

## Prerequisites

- Mac with **Xcode 16+** installed
- **Apple Developer Program** membership ($99/year)
- Apple ID enrolled at [developer.apple.com](https://developer.apple.com)

## Project structure

| Path | Purpose |
|------|---------|
| `ios/` | Native Xcode project (Capacitor shell) |
| `out/` | Static web build (generated by Next.js) |
| `resources/` | Source app icon and splash images |
| `capacitor.config.ts` | Capacitor settings |

**Bundle ID:** `com.hookai.app`  
**App name:** HookAI  
**Version:** 1.0

## 1. Build the web app and sync iOS

```bash
npm install
npm run build:ios
```

This runs `next build` and copies the static site into the Xcode project.

## 2. Open in Xcode

```bash
npm run ios:open
```

Or open manually: `ios/App/App.xcodeproj`

## 3. Configure signing in Xcode

1. Select the **App** target in the left sidebar
2. Open **Signing & Capabilities**
3. Select your **Team** (Apple Developer account)
4. Confirm **Bundle Identifier** is `com.hookai.app` (change if already taken)
5. Enable **Automatically manage signing**

## 4. Test on a device or simulator

1. Select a simulator or connected iPhone
2. Press **Run** (▶) or `Cmd + R`
3. Verify: topic input, generation, copy, theme toggle

## 5. Prepare App Store Connect

1. Go to [appstoreconnect.apple.com](https://appstoreconnect.apple.com)
2. **My Apps** → **+** → **New App**
3. Fill in:
   - **Platform:** iOS
   - **Name:** HookAI
   - **Primary Language:** English
   - **Bundle ID:** com.hookai.app
   - **SKU:** hookai-ios-001

## 6. Required metadata

### Privacy Policy URL (required)

Host `docs/PRIVACY_POLICY.md` via **GitHub Pages** (see below) and paste the URL in App Store Connect.

**Your URL after setup:** `https://haranilov.github.io/AI-App/PRIVACY_POLICY.html`

#### Enable GitHub Pages

1. Push the `docs/` folder to GitHub (`git push`)
2. Open [github.com/haranilov/AI-App/settings/pages](https://github.com/haranilov/AI-App/settings/pages)
3. **Build and deployment** → Source: **Deploy from a branch**
4. Branch: **main** → Folder: **/docs** → **Save**
5. Wait 1–2 minutes until the site is live
6. Open the URL above and confirm the policy renders

Example: `https://yourdomain.com/privacy`

### App Privacy questionnaire

In App Store Connect → App Privacy:

| Data type | Collected? | Notes |
|-----------|------------|-------|
| Contact info | No | — |
| User content | Yes | Video topics typed by user |
| Identifiers | No | — |
| Usage data | No | — |

**Third-party services the app contacts:**

- Puter.js (optional AI, may require login)
- Google Gemini / Groq / Pollinations / OpenAI (if user adds API keys)
- js.puter.com (script load)

Declare that data is **not linked to identity** and used for **app functionality**.

### Export compliance

`ITSAppUsesNonExemptEncryption` is set to `false` in Info.plist (HTTPS only, no custom encryption).

## 7. Archive and upload

1. In Xcode, set destination to **Any iOS Device (arm64)**
2. **Product** → **Archive**
3. When Organizer opens → **Distribute App**
4. **App Store Connect** → **Upload**
5. Follow the prompts (include symbols, automatic signing)

## 8. Submit for review

1. In App Store Connect, open the uploaded build
2. Complete all required fields (screenshots, description, privacy URL)
3. **Add for Review** → **Submit to App Review**

### Review tips

- Provide a **demo topic** in Review Notes: e.g. `how to make money online`
- Mention the app works without API keys (Puter + templates fallback)
- If Puter login appears, explain it in Review Notes

## Updating the app

After code changes:

```bash
npm run build:ios
npm run ios:open
```

Bump **Version** (user-facing) and **Build** number in Xcode before each upload.

## Replacing app icon or splash

1. Replace `resources/icon.png` (1024×1024) and `resources/splash.png` (2732×2732)
2. Run:

```bash
npm run assets:generate
npm run ios:sync
```

## Troubleshooting

| Issue | Fix |
|-------|-----|
| White screen on launch | Run `npm run build:ios` again |
| Signing errors | Select your Team in Signing & Capabilities |
| Bundle ID taken | Change `appId` in `capacitor.config.ts` and Bundle ID in Xcode |
| Module not found (dev) | `rm -rf .next && npm run dev` |
